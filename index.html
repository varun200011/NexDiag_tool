Capabilities

Built for real ECU
validation work

Covers every edge case your Visteon cluster validation programs require — Royal Enfield, TVS, Hero.

EOL
EOL Entry Sequence
Full 8-step entry: Ext → EOL session → security unlock → activation write → ECU reset → re-entry. Correct 10 03 → 10 60 after each negative test.
ISO-TP
Multi-Byte Writes
Auto ISO-TP first + consecutive frames for DIDs >4 bytes. All reqB4..reqB7 bytes correctly emitted for single-frame writes.
NRC
Full NRC Coverage
NRC 0x13 wrong length, 0x31 OOR above/below range, 0x33 no security, 0x7F wrong session, 0x7E sequence error — per DID, automatically.
2F
IO Control
Bit-map: ALL_OFF → each individual bit → ALL_ON. Range: min/25%/50%/75%/max. OOR below min and above max. Invalid ctrlParam 0x05.
SRD
Excel Import
Upload SRD Excel. Column mapper auto-detects I/O Control, Range, Description. Strikethrough rows excluded via openpyxl pre-processing.
Security
Mixed Algorithms
Level1 EXE + EOL formula combinations handled per security level. Live formula preview shows computed key before generating.
